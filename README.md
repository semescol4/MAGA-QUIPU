# MAGA-QUIPU — Solución del sistema (Release)

Este repositorio se utiliza como contenedor de entregables.

✅ **La solución completa del sistema está publicada en la sección _Releases_** (archivo ZIP).
- Descargar desde: **Releases → Latest**


## Contenido del ZIP
- Solución completa del sistema (fuentes + estructura original)
- Documentación / manuales (si aplica)

## Instalación / uso
1. Descargar el ZIP desde Releases
2. Descomprimir en una carpeta local
3. Abrir la solución / proyecto según corresponda

## Notas
- El ZIP se publica en Releases porque supera el límite de 100MB para archivos dentro del repositorio.
